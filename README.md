An R package to download [PanelApp](https://panelapp.extge.co.uk/) information.


Install
--------------------------------------------------------------------------------

    install.packages ("devtools")
    library (devtools)
    setRepositories (ind=1:2)
    install_github ("dmontaner/panelappR/pkg")
